# Atomatas_de_Pila_PDA
Este proyecto consiste en una aplicación de escritorio interactiva desarrollada en **Python** utilizando **Flet** como framework de interfaz gráfica. Permite el diseño, carga, simulación paso a paso y visualización de Autómatas de Pila tanto deterministas como no deterministas.

Estos pasos le ayudaran a ejecutar la aplicación en su entorno local:

### 1. Requisitos Previos
  - tener instalado **Python 3.8 o superior** y el software de trazado de gráficos **Graphviz** en su sistema operativo.
    *Nota para Windows:* Al instalar Graphviz, asegúrese de marcar la opción *"Add Graphviz to the system PATH for all users"* para que la generación de grafos funcione correctamente desde el script.

### 2. Instalar las Dependencias
Abra su terminal o línea de comandos dentro de la carpeta raíz del proyecto y ejecute el siguiente comando para instalar las librerías necesarias (`flet` y `graphviz`):
`pip install flet graphviz`

### 3. Ejecutar la interfaz
Una vez instaladas las dependencias, inicie el simulador ejecutando el script principal de la interfaz de usuario:
`python Prueba.py`
